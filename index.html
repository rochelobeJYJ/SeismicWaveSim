지진파 속도 · 주시곡선 공식
Vp√((K + 4μ/3) / ρ)
Vs√(μ / ρ)
Snellsin θ₁ / V₁ = sin θ₂ / V₂
직접파t = √(x² + h²) / V₁
반사τ²ᵢ = τᵢ² + (x/Vᵢ)²
임계t = τᵢ + x / Vₕ (Vₕ > Vᵢ)
반사계수R = |ΔV / ΣV|

K = 체적계수, μ = 전단계수, ρ = 밀도. 반사계수는 원본 C++ 구현과 동일한 속도 기반 근사식(R ≈ (V₂−V₁)/(V₂+V₁))입니다. 실제 지각에서는 임피던스 ρV 기반의 Fresnel 계수를 씁니다.
